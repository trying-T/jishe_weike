# AlgoLens

一个面向教学演示的 AI 推荐算法认知平台，用交互式页面帮助学习者理解推荐系统如何形成用户画像、如何影响内容分发，以及“信息茧房”“算法偏见”这类现象是怎样一步步出现的。

## 项目简介

本项目基于 `React + Vite + Tailwind CSS` 构建，围绕“推荐算法素养教育”设计了完整的学习路径。用户可以先了解概念，再进入模拟器进行点击交互，随后通过图表观察推荐结果如何变化，最后用学习模块和测验巩固知识。

适合用于：

- 课堂展示与课程作业
- 推荐系统基础认知教学
- AI 素养、媒介素养、算法偏见主题展示
- 前端交互式教学网站案例

## 核心功能

### 1. 首页导览

- 介绍平台定位和学习路径
- 提供模块入口与整体导航

### 2. 教学概览

- 解释推荐系统为什么存在
- 介绍用户画像、相似度计算、召回/排序/重排等核心概念
- 引出信息茧房、算法偏见等社会影响

### 3. 算法模拟器

- 模拟短视频推荐流的生成逻辑
- 点击不同类别内容后，系统会即时更新用户兴趣分布
- 展示“用户画像”“茧房指数”“推荐解释”“行为时间线”等反馈
- 用交互方式演示推荐系统如何逐渐收缩内容视野

### 4. 实时可视化

- 将模拟器中的交互结果转成图表
- 展示类别分布、兴趣占比、多样性趋势、最近交互记录
- 帮助学习者从数据角度理解推荐结果变化

说明：
可视化页面依赖模拟器写入的本地数据，因此建议先访问“算法模拟器”并完成几次交互，再进入“实时可视化”页面查看结果。

### 5. 学习模块

- 采用章节式内容组织
- 覆盖推荐系统背景、核心机制、系统流程与社会影响
- 支持展开阅读和弹窗深入学习

### 6. 知识测验

- 包含选择题、判断题、填空题与简答题
- 用于检验学习效果并强化关键概念

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发环境

```bash
npm run dev
```

启动后在浏览器打开本地开发地址即可预览项目。

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览构建结果

```bash
npm run preview
```

## 可用脚本

- `npm run dev`：启动开发服务器
- `npm run build`：构建生产包
- `npm run preview`：本地预览构建结果

## 项目特点

- 用交互代替静态讲解，更适合教学场景
- 页面结构完整，包含导览、学习、实验、可视化、测验闭环
- 适合作为课程作品、展示项目或 AI 素养教育网站原型

## 开发说明

- 当前项目为纯前端应用，无后端服务依赖
- 模拟器与可视化之间的数据通过浏览器 `localStorage` 共享
- 推荐结果与卡片内容为教学演示用途，不对应真实平台数据
